**Sales Analysis Dashboard – Power BI**

**Project Overview**

This project presents an interactive Sales Analysis Dashboard built using Microsoft Power BI. The dashboard analyzes sales performance and provides insights into revenue trends, product performance, and business growth.

The data was first cleaned and prepared in Microsoft Excel, then modeled and visualized in Power BI to create meaningful business insights.

**Objectives**

Analyse overall sales performance

Identify top-performing brands

Track sales trends over time

Compare performance across different categories with brand

Support data-driven business decisions

**Project Tools**

Microsoft Excel – Data Cleaning & Preparation

Microsoft Power BI – Data Modeling

DAX – Creating Measures and Calculations

Power BI Visualizations – Dashboard Design

**Dashboard Features**

Sales KPI Overview(Total Customers , Revenue,Cost and Profit)

Sales trend analysis over time

Top-selling brands

Category-wise performance

Interactive filters and slicers
